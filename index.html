gradle console

Easily interact with your Groovy and Java applications

Download .zip Download .tar.gz View on GitHub

This plugin allows you to easily interact with your Groovy and Java applications. Just run gradle console and you will get a console window with all your runtime dependencies loaded for you. This is useful in case you need to interact your project dependencies or your source code in a fast and flexible way.

Instalation

Just add the following snippet of code to your build.gradle file

buildscript {
    repositories {
        mavenCentral()
    }
    dependencies {
        classpath group: 'carlosgsouza', name: 'gradle-console', version: '1.0'
    }
}

Usage

Invoke the console task using gradle

gradle console
